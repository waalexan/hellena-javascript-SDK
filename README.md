# hellena-JavaScript-SDK

A JavaScript SDK for the Hellena platform.

## Installation

```bash
npm install hellena-javascript-sdk
```

## Usage

```javascript
const Hellena = require('hellena-javascript-sdk');

const client = new Hellena({
    apiKey: 'your-api-key'
});
```

## Features

- Easy integration
- Full API support
- TypeScript ready
